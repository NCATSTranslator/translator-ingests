# Ingest Documentation

This is a documentation folder for the aggregation of supplemental ingest-specific documentation.

Unlike the ingest-specific Reference Ingest Guides (RIGs), documentation generated
for internal reference about specific knowledge sources and their ingest code can be stored, 
without complaints from codespell, ruff and other linters. For example, if an _ad hoc_ exploratory data analysis
is performed on specific knowledge data sources, to guide ingest design, it can be kept here for future reference.
